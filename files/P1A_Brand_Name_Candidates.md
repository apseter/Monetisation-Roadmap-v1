## SCREENING CHECKLIST (Run on final 3-5 candidates)

### Step 1: Trademark Search (~30 min per name)
- [ ] EUIPO eSearch plus: https://euipo.europa.eu/eSearchCLW/ → Search Class 25 → exact + similar
- [ ] UKIPO: https://trademarks.ipo.gov.uk/ipo-tmtext → Search Class 25
- [ ] WIPO Global Brand Database: https://branddb.wipo.int/ → All jurisdictions
- [ ] USPTO TESS: https://tmsearch.uspto.gov/ → Basic word mark search

### Step 2: Domain Check (~5 min per name)
- [ ] [name].com — available? Price?
- [ ] [name].co.uk — available?
- [ ] [name].fr — available?
- [ ] [name].run — available?
- [ ] [name].co — available?
- [ ] Use Namecheap.com or Porkbun.com for availability + pricing

### Step 3: Social Handle Check (~5 min per name)
- [ ] Namechk.com: check all platforms simultaneously
- [ ] Instagram: @[name] or @run[name] or @[name]run
- [ ] TikTok: @[name]
- [ ] Strava Club: search clubs for "[name]"
- [ ] YouTube: youtube.com/@[name]

### Step 4: Pronunciation + Association Check (~15 min per name)
- [ ] Say it to 3 English speakers — can they spell it after hearing it once?
- [ ] Say it to 2 French speakers — same test
- [ ] Google "[name] meaning" — any negative associations?
- [ ] Google "[name] slang" — any profanity in any language?
- [ ] Translate in: French, German, Spanish, Urdu, Arabic — any negatives?

### Step 5: Runner Testing (~1-2 hours for 5-10 runners)
- Show name without context: "What does this make you think of?"
- Rate 1-5: How premium does this feel?
- Rate 1-5: How memorable is this?
- Rate 1-5: Would you trust a running brand with this name?
- "If you saw this on a pair of trail shorts, what price range would you expect?"

# P1A ADDENDUM: BRAND NAME SELECTION FRAMEWORK (Critical Gap Fix D01)

> **This section was added to close Critical Gap D01: Brand name not selected — blocks trademark, domain, Shopify, ALL marketing, packaging. Every week the name remains unselected delays the ENTIRE project by one week.**

## P1A.X1: 5-Step Name Selection Process (Complete in 1 Week)

**Step 1 (Day 1): Shortlist to 5 names.** Review candidates above. Pick your personal top 5 based on gut + SMILE scores.

**Step 2 (Day 2): Availability screening (2 hours).** For each name, check:
- .com domain (Namecheap)
- .co.uk domain
- Instagram handle
- TikTok handle
- UKIPO trademark search (gov.uk/search-for-trademark) — no matches in Class 25
- EUIPO search (euipo.europa.eu/eSearch) — no matches
- Google search — no major existing brand

**Eliminate any name that fails .com domain OR trademark availability.**

**Step 3 (Day 3-4): Test with 5 runners.** Share 2-3 names with trail runners and ask: most memorable? Sounds like a trail brand? Comfortable wearing? Negative associations?

**Step 4 (Day 5): DECIDE.** No perfect name exists. Many good-enough names do. The best name is the one you commit to.

**Step 5 (Day 5-7): Secure immediately.** Within 24 hours: purchase domains, register all handles, file UKIPO (£170), file EUIPO (€850 or defer 30 days). Total: £200-250 + £170 UKIPO.
