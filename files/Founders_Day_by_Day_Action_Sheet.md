## PHASE 1: VALIDATION & FOUNDATIONS (Days 1-14)

### DAY 1 — Capital & Name Shortlist
- [ ] Complete capital assessment worksheet (P7A Addendum) — determine Scenario A/B/C/D
- [ ] Based on scenario: set Run 1 size (150/200/250/300 units)
- [ ] Review P1A brand name candidates — shortlist your personal top 5
- [ ] Write down your top 5 names on paper
- **Time: 2 hours**

### DAY 2 — Name Screening & UK Ltd
- [ ] Screen all 5 names: check .com domain (Namecheap), .co.uk, Instagram, TikTok, UKIPO trademark search, EUIPO search, Google search
- [ ] Eliminate any name that fails .com or trademark
- [ ] Begin UK Ltd incorporation online (gov.uk/set-up-limited-company) — takes 1-2 days to process
- [ ] Apply for EORI number (gov.uk) — needed for imports
- **Time: 3-4 hours**

### DAY 3 (Saturday) — Customer Validation
- [ ] Go to parkrun wearing running gear. Volunteer or run. After the run, approach 5-7 trail runners
- [ ] Use the interview script from Gap Fix B15 — 10 minutes per person
- [ ] Record responses in the template (name, current shorts, pain point, concept reaction, price reaction, intent 1-10, email)
- [ ] Capture emails for your waitlist
- **Time: Half day (3-4 hours including travel)**

### DAY 4 (Sunday) — More Validation + Name Testing
- [ ] Conduct 3-5 more interviews via Instagram DM, friends who run trails, or local running club
- [ ] Show your top 2-3 brand names to interviewees — which is most memorable?
- [ ] Tally all responses — apply GO/PIVOT/STOP framework
- [ ] **DECISION: GO / PIVOT / STOP** — if STOP, save your money and stop here
- **Time: 2-3 hours**

### DAY 5 — Brand Name Decision + Digital Securing
- [ ] **SELECT YOUR BRAND NAME** — highest scoring name with all checks passed
- [ ] Purchase .com domain (Namecheap, ~£10)
- [ ] Purchase .co.uk domain (~£5)
- [ ] Register Instagram handle
- [ ] Register TikTok handle
- [ ] Create Strava Club (upload logo placeholder, add trail cover image)
- [ ] Register Twitter/X handle (defensive)

- **Time: 2-3 hours | Cost: ~£25**

### DAY 6 — Trademark + VAT + Banking

- [ ] Apply for voluntary UK VAT registration (gov.uk — takes 2-8 weeks to process)
- [ ] Open Wise Business account (wise.com/business — verification takes 1-3 weeks)

## PHASE 2: PRODUCT DEVELOPMENT (Days 15-140)

### DAYS 15-21 — Lock Suppliers + Order Samples
- [ ] Select top 2 Sialkot factories (scoring 20+/30)
- [ ] Select top 2-3 Chinese fabric suppliers (scoring 25+/35)
- [ ] Order 1-metre fabric samples from top 2-3 suppliers ($30-80 total)
- [ ] Commission tech pack designer on Upwork ($200-350)
- [ ] Brief designer with product specs from P2C
- [ ] ⏳ Continue daily social media posting (5x/week Instagram, 4x/week TikTok)
- [ ] ⏳ Continue Reddit and Facebook group engagement
- **Cost: $230-430**

### DAYS 22-35 — Fabric Samples Arrive + Tech Pack
- [ ] Inspect fabric samples: hand feel, GSM, stretch, colour
- [ ] Test DWR: spray water at 45° angle — does it bead and roll off?
- [ ] Apply Quality Gate G1: sample matches spec? (P2B Addendum X3)
- [ ] If PASS: lock primary supplier. Order counter-sample fabric (5-10 metres).
- [ ] If FAIL: order from backup supplier. Do NOT compromise on DWR.
- [ ] Tech pack should be nearing completion — review with designer
- [ ] ⏳ Publish first 2 blog posts (SEO content from P5A.16)
- [ ] ⏳ Waitlist target: 50-100 signups by end of Week 5

### DAYS 36-50 — Pattern Development + Counter Samples
- [ ] Send approved tech pack + fabric to selected Sialkot factory
- [ ] Begin pattern development (2-3 weeks)
- [ ] Commission patterns for BOTH outer and liner simultaneously
- [ ] Begin snap mechanism prototyping — order Prym or YKK snap hardware
- [ ] ⏳ Start building relationships with micro-influencers (comment on posts, engage)
- [ ] ⏳ Identify 30-50 micro-influencer targets for later seeding

### DAYS 51-80 — Counter Sampling (2-3 Rounds)
- [ ] Receive Counter Sample Round 1 — inspect: stitching, snap placement, pocket size, overall construction
- [ ] Provide detailed feedback to factory
- [ ] Counter Sample Round 2 — should address Round 1 issues
- [ ] **WEEK 9 SNAP DECISION GATE (Day ~63):** Do snaps work after 2 rounds?
  - [ ] If YES: continue with snap system
  - [ ] If NO: **PIVOT to Velcro attachment immediately** — don't waste more time on snaps
- [ ] Counter Sample Round 3 (if needed) — final construction refinements
- [ ] ⏳ Continue social media (should be approaching 300-500 Instagram followers)
- [ ] ⏳ Waitlist target: 100-150 signups

### DAYS 81-120 — Fit Sampling (2-3 Rounds)
- [ ] Fit Sample Round 1 — try on yourself + send to 3-5 testers (different body types)
- [ ] Collect fit feedback: waist, hip, inseam, liner compression, phone pocket bounce
- [ ] Fit Sample Round 2 — revised patterns based on feedback
- [ ] Test compression: compare to a known medical garment using P2C wrapped-tube method
- [ ] Test phone pocket: load iPhone 16 Pro Max in case, run 5km, assess bounce
- [ ] Run snap stress test: 200 attach/detach cycles (P2C)
- [ ] Fit Sample Round 3 (if needed) — final fit refinements
- [ ] ⏳ Order bulk fabric from China at Fit Sample Round 1 approval (~Day 95)
- [ ] ⏳ Begin product photography with fit samples (for website and marketing)
- [ ] ⏳ Start building Shopify product pages (descriptions, images, pricing)
- [ ] ⏳ Waitlist target: 150-200 signups

### DAYS 121-140 — Pre-Production + Final Approval
- [ ] Pre-production sample submitted by factory
- [ ] Inspect PP sample against approved fit sample — must match exactly
- [ ] **PP APPROVAL: GO/NO-GO for production**
  - [ ] If APPROVE: authorise bulk cutting
  - [ ] If REJECT: one more round (add 2-3 weeks)
- [ ] Finalise packaging: order kraft mailers, branded stickers, insert cards (£90-200)
- [ ] Order care labels and hang tags from factory
- [ ] Design and order insert card (A6, double-sided with QR codes)
- [ ] ⏳ Start Shopify Markets configuration (UK, EU, USD)
- [ ] ⏳ Configure shipping zones and rates in Shopify
- [ ] ⏳ Set up Sendcloud for label printing
