## P2C.1: TECH PACK CREATION

### P2C.1.1: What a Professional Activewear Tech Pack Contains

Your tech pack is the single most important document in this entire process — it eliminates ambiguity between you and your factories. Each tech pack needs 11 core sections:

| Section | Contents | Why It Matters |
|---|---|---|
| 1. Cover page | Style number, version control, date, designer | Prevents confusion between revisions |
| 2. Design overview | Performance features, brand positioning, target customer | Context for factory |
| 3. Flat sketches | Front, back, inside views + detail views of pockets, snaps, waistband | The visual blueprint — most referenced page |
| 4. Construction details | Stitch types per seam, SPI (stitches per inch), seam allowances | Defines quality standard |
| 5. Bill of Materials (BOM) | Every component with supplier, spec, quantity per garment | Costing and procurement |
| 6. Colorway page | Pantone references for each color, fabric + trim matching | Color consistency |
| 7. Size specification sheet | All points of measure with tolerances per size | Fit accuracy |
| 8. Grading page | Increment tables showing how each measurement changes per size | Multi-size production |
| 9. Trim details | Snaps, drawcord, elastic, zippers — exact specs and placement | Hardware consistency |
| 10. Labels & packaging | Care label content (EN+FR), brand label, hang tag, fold method, polybag | Compliance + branding |
| 11. Modular assembly doc | How outer + liner connect — snap placement diagram with exact measurements | UNIQUE to our product |

### P2C.1.2: Construction Specifications

| Component | Seam | Stitch Type | SPI | Notes |
|---|---|---|---|---|
| Outer short — side seams | 4-thread overlock | Type 504 | 12-14 | Standard athletic construction |
| Outer short — hem | Coverstitch | Type 406 | 10-12 | Clean finish, allows stretch |
| Outer short — waistband | Lockstitch + overlock | Type 301/504 | 10-12 | Structural — must hold elastic + drawcord |
| Outer short — pocket | Lockstitch | Type 301 | 12-14 | Precision needed for zipper installation |
| **Liner — ALL body seams** | **Flatlock** | **Type 607** | **14-16** | **Critical: zero-ridge seams against skin** |
| Liner — leg opening | Coverstitch or flatlock | Type 406/607 | 12-14 | Comfort at thigh contact |
| Liner — waistband | Overlock + coverstitch | Type 504/406 | 10-12 | Under outer short, less skin-critical |

### P2C.1.3: Tools for Creating Tech Packs

| Tool | Cost | Best For |
|---|---|---|
| Adobe Illustrator | $23/month | Industry-standard flat sketches. All factories accept AI/PDF/EPS. |
| Google Sheets | Free | BOM tables, spec sheets, grading charts |
| Style3D | Free (2025+ free tier) | 3D garment visualization — show factory the vision |
| Techpacker | $35-95/month | Drag-and-drop card system, manufacturer collaboration portal |

**Recommended stack:** Adobe Illustrator ($23/mo) for sketches + Google Sheets (free) for specs + Style3D (free) for 3D visualization = **$23/month total.**

## P2C.3: PATTERN DEVELOPMENT FOR STRETCH PERFORMANCE FABRICS

### P2C.3.1: Critical Difference: Outer Short vs Liner

| Attribute | Outer Short (Ripstop) | Compression Liner (Nylon/Spandex) |
|---|---|---|
| Ease type | **Positive ease** (pattern larger than body) | **Negative ease** (pattern smaller than body) |
| Stretch reliance | Minimal — ripstop has limited stretch | Full — fabric stretch creates fit + compression |
| Pattern approach | Standard shorts pattern with ease | Compression engineering with calculated reduction |
| If fabric changes | Minor adjustment | **ENTIRE pattern must be re-drafted** |

### P2C.3.2: Negative Ease for Compression Liner

Formula: **Finished garment measurement = Body measurement × (1 - negative ease %)**

| Body Zone | Negative Ease % | Example (24" thigh) | Pattern Measurement |
|---|---|---|---|
| Waistband | 5-8% | 32" waist → | 29.4-30.4" |
| Hip | 8-12% | 38" hip → | 33.4-35.0" |
| Upper thigh | 10-15% | 24" thigh → | 20.4-21.6" |
| Mid-thigh | 8-12% | 22" thigh → | 19.4-20.2" |
| Leg opening | 5-8% | 20" opening → | 18.4-19.0" |

**Critical rule:** If the fabric changes, the pattern must be completely re-drafted. A pattern engineered for 75% stretch fabric will NOT work with 50% stretch fabric. **Lock your exact production fabric BEFORE finalizing patterns.**

### P2C.3.3: Women's Patterns — Independently Developed

Women's patterns MUST be developed independently, NOT scaled down from men's:

| Attribute | Men's | Women's | Why It Matters |
|---|---|---|---|
| Hip:waist ratio | 0.85-0.95 | 0.70-0.80 | Much larger difference in women's — scaling down creates too-tight hips or too-loose waist |
| Back rise | Standard | +1-2" higher | Accommodates greater hip curvature |
| Crotch curve | Shallower | Deeper, more curved | Anatomically different — forcing men's curve on women's = poor fit |
| Front rise | Standard | Higher (women prefer higher-rise waistband) | Comfort + coverage preference |
| Thigh angle | Narrower | Wider (relative to hip) | Different leg angle at hip joint |

### P2C.4.2: Tolerances

| Measurement Type | Tolerance |
|---|---|
| Critical circumferences (waist, hip) | ±½" (13mm) |
| Lengths (inseam, rise) | ±¼" (6mm) |
| **Snap placement** | **±⅛" (3mm) MAXIMUM** |
| Leg opening | ±¼" (6mm) |

**Snap placement tolerance is the tightest in the entire spec** — misalignment between outer and liner snaps prevents proper attachment. Tolerance must be less than half the grade increment.

## P2C.5: SAMPLE DEVELOPMENT — 5 STAGES

### P2C.5.1: Stage Progression

| Stage | Purpose | Fabric | Rounds | Cost per Sample | Timeline |
|---|---|---|---|---|---|
| **1. Proto sample** | Basic shape, proportions, pocket placement | Substitute OK | 1-2 | $30-60 | 1-2 weeks each |
| **2. Fit sample** | On-body fit, movement, seam placement | Actual fabric required | 2-3 | $75-150 | 1-2 weeks each |
| **3. Size set** | Verify grading across all sizes | Actual fabric | 1 | $150-300 (one per size) | 2-3 weeks |
| **4. Pre-production (PP)** | Final production-representative unit | All final materials | 1 | $75-150 | 1 week |
| **5. Top of Production (TOP)** | Verify consistency during bulk run | Production line pull | 1 | $0 (part of run) | During production |
| **TOTAL** | | | **6-9 rounds** | **$500-1,100** | **12-20 weeks** |

### P2C.5.2: What Changes at Each Stage

**Proto → Fit:** All measurements adjusted based on mannequin/body evaluation. Pocket placement finalized. Seam construction upgraded from substitute to production methods.

**Fit → Fit (revision):** Specific corrections — "increase back rise by 0.5 inch," "move phone pocket zipper start point down 1cm," "reduce leg opening by 0.5 inch." This is the stage with highest resubmission rate — expect 2-3 rounds.

**Fit → Size Set:** Approved fit sample in base size (M) is graded to all production sizes. Each size checked against spec sheet.

**Size Set → PP Sample:** Made on actual production line (or simulated production conditions) with ALL final materials — production fabric, production trims, production labels, production packaging.

**PP → TOP:** Random pull from first 5-10 units of bulk production to verify the line is replicating the PP sample consistently.

## P2C.6: FIT TESTING PROTOCOL FOR TRAIL RUNNING SHORTS

### P2C.6.1: Static Fit Checks

| Test | What to Check | Issue If Failed |
|---|---|---|
| Stand naturally | Waistband sits at natural waist, no roll, no dig | Rise too short or waistband too narrow |
| Full squat (10 reps) | Back rise coverage, crotch restriction, waistband ride-down | Back rise too short, crotch too shallow |
| Forward lunge (each leg, 5 reps) | Inseam restriction, liner ride-up, outer-liner separation | Inseam too tight, negative ease too high |
| Lateral lunge | Side seam pull, leg opening restriction | Leg opening too narrow |
| High knee (30 sec × 3 sets) | Front rise adequacy, waistband stability | Front rise too short |
| Seated position (60 sec) | Waistband compression on stomach, liner dig at thigh | Waistband too rigid, liner ease too high |

### P2C.6.2: Dynamic Running Tests

| Test | Duration | Pace | What to Check |
|---|---|---|---|
| Walk | 5 min | 3 mph | Baseline comfort, waistband position, no immediate issues |
| Easy jog | 10 min | 5 mph | Overall bounce, liner behavior, initial chafe detection |
| Tempo run | 10 min | 7 mph | Phone pocket stability, snap attachment stress, fabric response |
| Hill climbs | 5 min | Steep grade | Back panel coverage, snap stress during extreme hip flexion |
| Descent | 5 min | Technical | Phone pocket security during jarring impact |
| **Total test run** | **30+ min** | **Mixed** | **Comprehensive evaluation** |

### P2C.6.3: Chafe Point Documentation

1. Apply cornstarch to suspected friction zones BEFORE test run
2. Run for 30, 60, and 90 minutes
3. Photograph redness at each interval
4. Mark on standardized body diagram
5. Rate severity: 0 (none) / 1 (slight pink) / 2 (red) / 3 (raw) / 4 (broken skin)

**Common chafe zones in running shorts:** inner thigh (bilateral), groin crease, waistband sides, leg opening edge, phone pocket zipper pull area.

## P2C.7: PHONE POCKET BOUNCE TESTING

### P2C.7.1: Test Protocol

Use a **200g weight** (approximate smartphone weight) in the phone pocket.

| Test Condition | Speed | Grade | Acceptable Bounce |
|---|---|---|---|
| Walking | 3 mph | Flat | <0.5" vertical displacement |
| Easy jog | 5 mph | Flat | <0.75" vertical displacement |
| Tempo run | 7 mph | Flat | <1.0" vertical displacement |
| Sprint | 9+ mph | Flat | <1.5" vertical displacement |
| Descent | 5 mph | -15% grade | <1.5" vertical displacement |

### P2C.7.2: How to Measure

Film from the side with phone camera on tripod at waist height. Play back in slow motion. Measure highest bounce point relative to the pocket resting position. If bounce exceeds threshold at any speed, adjust pocket depth, width, opening width, or add internal phone-shaped panel to reduce movement.

## P2C.8: SNAP ATTACHMENT STRESS TESTING

### P2C.8.1: Test Protocol

| Test | Method | Pass Criteria |
|---|---|---|
| **Static load** | Pull 15 lbs (6.8 kg) force on each snap for 30 seconds | Zero snaps open |
| **Dynamic running** | 30+ minute trail run with integrity check every 10 minutes | Zero snaps loosen, no liner shift |
| **200-cycle endurance** | Attach/detach 200 times. Measure pull force at 0, 50, 100, 150, 200 cycles | Pull force drops <20% from initial |
| **Directional stress** | Pull laterally, vertically, diagonally on each snap | Zero failure in any direction |
| **Multi-angle** | Simulate running motion (hip flexion, rotation, bounce) for 5 minutes | Zero accidental detachment |
| **Wash durability** | Machine wash cold, tumble dry low. Test after 10 and 25 cycles | Pull force maintained, no corrosion |

### P2C.8.2: Documentation

Record pull force (in kg or lbs) at each interval using a kitchen scale or spring scale. Photograph snap hardware at 0, 100, and 200 cycles for deformation. If any snap fails any test, re-evaluate snap brand, size, or placement.

## P2C.9: COMPRESSION LINER ENGINEERING (15-20 mmHg)

### P2C.9.1: Target Fabric Specification

| Attribute | Specification | Why |
|---|---|---|
| Composition | 78-82% Nylon / 18-22% Spandex | Industry standard for moderate athletic compression |
| Weight | 220-240 GSM | Sweet spot for 15-20 mmHg |
| Stretch | Minimum 30-40% elongation, 4-way | Enables negative ease engineering |
| Recovery | >90% (ideally >95%) | Below 90% = garment loses compression over time |
| Inner face | Brushed | Moisture wicking + comfort against skin |
| Durability | >50,000 abrasion cycles | Long-term performance |

### P2C.9.2: Uniform vs Graduated Compression

**Uniform compression recommended for shorts liner.** Graduated compression (highest at extremity, decreasing toward heart) is designed for below-knee garments to aid venous return. For the short compression zone of a thigh-length liner, the primary benefit comes from muscle stabilization — reducing oscillation and vibration during running — not circulatory improvement. Uniform 15-20 mmHg across the thigh achieves this most effectively.

### P2C.9.3: Verification Methods

| Method | Cost | Accuracy | When to Use |
|---|---|---|---|
| **Comparative method** | $30-50 (reference garment) | Good estimate | All sample rounds |
| **Sphygmomanometer (modified)** | $20-30 (equipment) | Approximate | Prototype validation |
| **Lab test (Hohenstein/SGS/Intertek)** | $200-1,000 | Certified precise | Before making mmHg marketing claims |

**Comparative method:** Buy known 15-20 mmHg medical compression shorts (JOBST or Sigvaris). Cut equal-width strips. Stretch both to same %. Compare force required — should feel similar. Have 5+ testers wear your liner and the reference simultaneously (one per leg) and rate 1-10.

## P2C.10: ANTI-CHAFE PANEL & GRIPPER ELASTIC ENGINEERING

### P2C.10.1: Anti-Chafe Zones

| Zone | Engineering Solution |
|---|---|
| **Inner thigh** (highest priority) | Position ALL seams AWAY from inner thigh. Use smooth nylon-face fabric. Add diamond gusset. |
| **Crotch point** | Diamond gusset: 3" × 6" (men's), 3.5" × 7" (women's) — eliminates problematic center seam |
| **Waistband edge** | Fold over or bond waistband edge — no raw edge contact |
| **Leg opening** | Silicone gripper elastic prevents ride-up without constriction |

### P2C.10.2: Silicone Gripper Elastic Application

**Use 1.5-2" wide silicone-dotted gripper elastic, sewn application** for initial Sialkot production. This provides excellent grip without the tourniquet effect of continuous silicone bands.

Laser cutting is available in Sialkot for liner edges — recommended for the modern, clean raw-edge aesthetic used by premium compression brands.

## P2C.12: CUTTING, SEWING & PRODUCTION EXECUTION

### P2C.12.1: Fabric Relaxation (Non-Negotiable)

| Fabric | Minimum Relaxation Time | Method |
|---|---|---|
| Compression nylon/spandex (liner) | **24 hours minimum** | Unroll, lay flat on breaker trolley, room temperature |
| Ripstop nylon (outer) | **4-8 hours minimum** | Unroll, lay flat, allow tension to release |

If before/after dimensions differ by >0.6%, extend relaxation overnight.

### P2C.12.2: Cutting

- Target **80-85% marker efficiency** (how much of the fabric becomes product vs waste)
- Cut stretch fabric in **low-ply layers (20-30 max)** — high-ply stacks cause lower layers to distort
- Use weights, NOT pins, to hold stretch fabric — pins create holes and distortion
- For ripstop: consider hot cutting with heated blade to seal edges (prevents fraying)

### P2C.12.3: Production Line Setup (300 Units)

**Estimated team: 8-12 people**

| Station | Machine | Operator | Operation |
|---|---|---|---|
| 1 | Cutting table + rotary cutter | 1 cutter | All fabric cutting |
| 2 | Overlock (4-thread) #1 | 1 operator | Outer short side seams + inseam |
| 3 | Overlock (4-thread) #2 | 1 operator | Outer short — gusset, waistband prep |
| 4 | Lockstitch | 1 operator | Pocket construction, zipper installation |
| 5 | Coverstitch | 1 operator | Outer short hems, waistband finish |
| 6 | Flatlock | 1 operator | Liner all seams (outsource if not in-house) |
| 7 | Snap press | 1 operator | Snap installation on outer + liner |
| 8 | Heat press | 1 operator | Reflective tape, label application |
| 9 | QC station | 1 inspector | 100% measurement + visual inspection |
| 10 | Finishing + packing | 1-2 packers | Thread trimming, pressing, folding, bagging |

### P2C.12.4: Production Capacity

| Metric | Value |
|---|---|
| SAM per complete unit (outer + liner) | 18-25 minutes |
| Efficiency Day 1-5 (learning curve) | ~50% |
| Efficiency Day 6-10 | ~70-80% |
| Output Day 1-5 (8 operators) | 75-100 units/day |
| Output Day 6-10 | 110-140 units/day |
| **300 units complete** | **3-5 working days** (after learning curve) |
| Total production (cutting → packing) | **~3.5-4 weeks + 1-week buffer** |

### P2C.12.5: Pilot Batch First

**Run a pilot batch of 15-20 units BEFORE committing to the full 300.** This surfaces 80% of production issues before they affect entire inventory. Review pilot batch with full QC inspection. Fix any issues. Then proceed to bulk.

## P2C.13: QUALITY CONTROL DURING PRODUCTION

### P2C.13.1: Seven QC Checkpoints

| # | When | What | Pass Criteria |
|---|---|---|---|
| 1 | Incoming fabric | GSM verification, DWR spray test, shade matching | GSM ±5%. Water beads. No shade variation between rolls. |
| 2 | Pre-cutting | Fabric relaxation verified | Dimensions stable (change <0.6%) |
| 3 | Post-cutting | 5% of cut pieces checked against pattern | Within ±¼" of pattern |
| 4 | In-line (every 50 units) | Stitch density, seam alignment, thread tension, pocket placement | SPI within spec. No puckering. |
| 5 | **Snap pull test (100%)** | **Pull each snap with 15 lbs force for 10 seconds** | **Zero pull-outs. Zero fabric tear. #1 CRITICAL checkpoint.** |
| 6 | Measurement audit | 5 key measurements per garment vs spec sheet | All within tolerance |
| 7 | Final inspection (100%) | Zipper cycles (10x), elastic recovery, threads trimmed, stains, labels, symmetry, **needle detection** | Zero visible defects. Zipper smooth. No loose threads. No metal fragments. |

### P2C.13.2: Needle Detection

**Critical safety check.** Run every garment through a needle detector (or manually check with a strong magnet at multiple points) to ensure no broken needle fragments are embedded in the fabric. Broken needles during sewing are common with synthetic fabrics and can injure the end consumer.

## P2C.14: COMMON DEFECTS & PREVENTION

### P2C.14.1: Stretch Fabric Specific Defects

| Defect | Cause | Prevention |
|---|---|---|
| Skipped stitches | Wrong needle type for fabric | Use **ballpoint (SUK) needles** for compression knit, **Microtex/Sharp** for ripstop |
| Seam puckering | Thread tension too high | Reduce needle thread tension. Enable differential feed. Use core-spun poly thread. |
| Wavy seams | Feed differential incorrect | Set front feed slightly higher than rear on overlock/coverstitch |
| Fabric damage/holes | Sharp needle on knit fabric | Switch to ballpoint immediately |
| Stretching during sewing | Fabric pulled through machine | Use presser foot with reduced pressure. Never pull fabric. |
| Uneven stitch length | Inconsistent feed | Check feed dog condition. Maintain consistent speed. |
| Oil staining | Machine over-lubricated | Clean machine before production. Use minimal silicone lubricant. |
| Snap pull-through | Snap installed on thin fabric without reinforcement | Add interfacing or fabric reinforcement patch behind every snap point |

### P2C.14.2: Needle & Thread Reference

| Fabric | Needle Type | Needle Size | Thread |
|---|---|---|---|
| Ripstop nylon (outer) | Microtex/Sharp | 70/10 - 80/12 | Polyester core-spun Tex 40-60 |
| Nylon/spandex (liner) | Ballpoint/Jersey (SUK) | 75/11 - 90/14 | Wooly nylon (loopers) + core-spun poly (needle) |

**Never use cotton thread on synthetic fabrics.** Cotton shrinks differently, degrades faster, and creates uneven tension.

### P2C.14.3: Post-Sewing

- **No garment washing required** for synthetic fabrics (no enzyme/stone wash needed)
- Light steaming removes handling wrinkles
- **Keep temperature below 130°C / 265°F for nylon** (it melts at low temperatures)
- Use pressing cloth for any direct iron contact
- Avoid steam on snap hardware (prevents tarnishing)
