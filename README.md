## Quick Start

1. Open `index.html` in your browser (or visit the live site above)
2. Start with the **Founder's Playbook** (10-page executive summary)
3. Follow the **Day-by-Day Action Sheet** (177-day task sequence)
4. Reference specific subproject documents as you execute each phase

## How to Enable GitHub Pages

1. Go to **Settings** → **Pages**
2. Under "Source", select **Deploy from a branch**
3. Select **main** branch, root folder `/`
4. Click **Save**
5. Your site will be live in 1-2 minutes at `https://YOUR-USERNAME.github.io/trail-shorts-gtm/`
